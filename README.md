## ヒット&ブロー
ヒット&ブローは4色で作られた色列を当てるゲームです。  
1人でも複数人でも遊べますが、複数人でのデバックは行えていません。

`/hab book`でゲーム管理用の本を取得してください。ゲームへの参加やゲームの開始などはこの本で行えます。
